Patrones de diseño

Singleton

El patrón Singleton es un patrón de diseño creacional que garantiza que una clase tenga una única instancia y proporciona un punto de acceso global a esa instancia. Este patrón se utiliza para controlar la creación de objetos, limitando la instancia de una clase a una única instancia y proporcionando un método para acceder a esa instancia desde cualquier punto del programa.
